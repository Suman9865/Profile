Project

Worker Finder Application

Worker Finder" is a web-based platform designed to streamline the connection between daily paid workers and job providers.The project is built using the MERN stack (MongoDB, Express.js, React, Node.js), showcasing a modern and scalable architecture. The primary goal is to facilitate efficient worker enrollment and allow users to access worker details based on their city.The platform incorporates the Indian cities API to provide users with suggestions while searching for workers based on city names.

Doctor Appointment System

The Doctor Appointment System is a web-based application designed to streamline the process of scheduling and managing doctor appointments within a healthcare environment.The project is built using the Java, JSP, Servlet, HTML, CSS, SQL. It serves as a digital platform that connects patients with healthcare providers eliminating common challenges such as long waiting times and manual appointment booking.The system aims to improve the patient experience by providing a user-friendly platform for appointment scheduling.

Automated Email Manager

The goal of the project is to create an automated email manager using the Gmail API from Google Console. JavaScript programming language is used for managing emails.The system will identify unread emails, categorize them based on whether they have been viewed or not, and automatically respond to unseen emails with a predefined message. Obtain API credentials and configure authentication for access to Gmail accounts. Ensure secure authentication using OAuth 2.0 to access the Gmail API securely.
